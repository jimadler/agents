# Project Structure Reference
Standard directory layout for Agentic OS operations:
- ~/Projects/agents/hermes/ : Global project root.
- ~/Projects/agents/hermes/identity/ : Governance layer (VALUES.md, BOUNDARIES.md, TONE.md, EXPECTATIONS.md).
- ~/Projects/agents/hermes/projects/<project>/ : Individual initiative root.
- ~/Projects/agents/hermes/__data__/ : Persistent data archives (tar.gz backups).
